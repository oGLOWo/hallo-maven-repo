hallo-maven-repo
================

Public libraries that Hallo uses that we could not find a maven repo for

Put this in your pom.xml

```xml
<repositories>
    <repository>
        <id>hallo-maven-repo-releases</id>
        <url>https://github.com/halloinc/hallo-maven-repo/raw/master/releases</url>
    </repository>
</repositories>
```
